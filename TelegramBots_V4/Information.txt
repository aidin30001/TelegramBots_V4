/start - старт бота
/file - получение файла по его имени с расширением 
    пример: /file файл.расширение
    вводит через пробел
/allfile - получение списка скаченных файлов
/h - информация о командах
/deletefile - удаление файла по его имени с расширением
    пример: /deletefile файл.расширение
    вводит через пробел
/df - удаление файла по его имени с расширением
    пример: /df файл.расширение
    вводит через пробел
/rename - переименовывать уже существует файл на новый
    пример: /rename старый_файл.расширение новый_файл.расширение
    вводит через пробел
